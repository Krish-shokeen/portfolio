# Krish Shokeen - Portfolio

A modern, responsive portfolio website built with React.js featuring a gaming/tech aesthetic with smooth animations and interactive elements.

## 🚀 Live Demo

[View Portfolio](https://your-deployed-url.com)

## ✨ Features

- **Modern Design** - Dark theme with cyan, coral, and blue accent colors
- **Responsive** - Works seamlessly on desktop, tablet, and mobile
- **Smooth Animations** - Scroll animations, hover effects, and interactive elements
- **Gaming Aesthetic** - Glassmorphism cards, gradient text, and glowing effects

## 📂 Sections

- **Hero** - Introduction with typing animation and profile photo
- **About** - Personal info with animated stats counters
- **Education** - Academic timeline, certifications, and achievements
- **Skills** - Technology cards with hover effects
- **Projects** - Featured projects with links
- **Contact** - Contact form and social links
- **Footer** - Social media links including Buy Me a Coffee

## 🛠️ Tech Stack

- React.js
- CSS3 (Custom animations, Flexbox, Grid)
- Font Awesome Icons

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Krish-shokeen/portfolio.git

# Navigate to project directory
cd portfolio-react

# Install dependencies
npm install

# Start development server
npm start
```

## 🏗️ Build

```bash
npm run build
```
